## Important Note

Rotary cup equipment is not a universal fix for every coating defect. Good results depend on workpiece geometry, grounding, powder quality, powder delivery, booth airflow, spray distance, line speed, curing, and operator/process control. Use test spraying or site data before making final equipment decisions.
